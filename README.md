# Chess database tool

A fast and modern chess analysis and database tool written in Rust

## Compiling and testing

Standard `cargo` commands.

`cargo build/run` for debug builds, `cargo build/run --release` for release build.

`cargo test` to run all the tests. `cargo test test_function_name` for specific functions.

## Docs

Again, standard `cargo` commands.

`cargo doc` to build the docs. `cargo doc --open` to open in the browser.
